Welcome to Mangoes' SoftDev Project! Here are three cool maps you can use to learn more about education in the U.S.A!

Please, choose what you are interested in:


This project is built using choropleth and using data with each state's records on it. The information displayed on each map is total revenue of each state in 2016, the amount of students in each state in 2016, and the average NAEP math exam scores for the eighth grade from each state in 2017. We wanted to see how the financial state of the environment children lived in resulted in whether they went to school and how it would affect their performance on standardized exams.
